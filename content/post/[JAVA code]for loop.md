- variables in for statement will not keep memory after loop is finished
- break : finish the whole loop
- continue : skip last and move to next loop

- 0~9 loop
```java

for (int i = 0; i < 10; i++) {

}
```
- 10~0 loop
```java

for (int i = 10; i > -1; i--) {

}
```
- only read the arr, cannot modify
```java

for (String str : arr) { 

}
```
